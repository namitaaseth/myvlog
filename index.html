My First Console Project

Visual Studio Code is a source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git. Download Visual Studio Code. Create a new folder on your computer. Open Visual Studio IDE and click on File > Open Folder, then browse and open the folder created. Install Live-Server Extension.

To open the console screen right click on the screen and click on inspect. Click on Console. To print something on the console screen we use "console.log()".
